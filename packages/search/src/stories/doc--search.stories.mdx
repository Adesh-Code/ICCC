## API

For all supported MUI `<Autocomplete />` props, see [Autocomplete API](https://mui.com/material-ui/api/autocomplete/).

The Search component accepts an `options` prop, which is an array of strings used to populate the dropdown.

For all examples below, we provide the following array to the `options` prop:

```js
const autodeskProducts = [
  "3DS Max",
  "AutoCAD LT",
  "AutoCAD",
  "BIM 360",
  "Factory Design Utilities",
  "Fusion 360"
  "Inventer CAM",
  "Inventor Nastran",
  "Inventor Nesting",
  "Inventor",
  "Maya",
  "Revit"
];
```

## Autosuggest highlight

If you want to emphasize text in the suggested options, you can opt for the [autosuggest-highlight](https://github.com/moroshko/autosuggest-highlight) package.

For a better understanding of how this package works with the Autocomplete MUI component, take a look at the [Highlights](https://mui.com/material-ui/react-autocomplete/#highlights) example provided by Material UI.
