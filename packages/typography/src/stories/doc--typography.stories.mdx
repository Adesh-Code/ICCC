### Tabular numeric

The `tabularNumeric` prop displays a set of figures where numbers are all of the same size, allowing them to be easily aligned like in tables.

<Canvas>
  <Story name="Tabular numeric">
    {(args) => (
      <Typography {...args} tabularNumeric>
        $ 1234 <br /> $ 1564 <br /> $ 5321 <br /> $ 4789 <br />
      </Typography>
    )}
  </Story>
</Canvas>

### Dark theme

## Accessibility

Typography patterns should be added to the page in an informational hierarchy, with no skipping of heading levels. Use correct semantic HTML tags for your content hierarchy needs to achieve this.
