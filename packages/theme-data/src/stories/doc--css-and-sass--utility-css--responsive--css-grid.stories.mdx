# Grid utility CSS

Digital HIG utility CSS provides a collection of CSS grid classes based on [Tailwind CSS](https://tailwindcss.com/).

For more in-depth information about CSS grid, [MDN basic concepts of grid layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout/Basic_Concepts_of_Grid_Layout) is a good place to go.

Note: Requires utility CSS to be present in the document. See [the CSS overview page](?path=/docs/iccc-css-and-sass-css-overview--page) for more details.

## Digital HIG ready

Utility CSS is responsive-design ready. The available classes are customized to match the breakpoint Figma specification, so refer to the [grid tokens page](/docs/iccc-theme-data-tokens-grid--page) to consult the available grid tokens, and [grid primitives page](/docs/iccc-theme-data-primitives-grid--page) for the full suite of grid-related options.

## How to use Tailwind CSS

Utility classes are generated with a `dhig-` prefix. Please use the Tailwind documentation and adjust classnames accordingly.

For example, in Tailwind docs:

`grid` display: grid;

Becomes:

```html
<div class="dhig-grid">This element will have display set to grid</div>
```

Each row is split in 3 equal-sized cells in small screens, 4 in medium and 6 in large.

### Cell size and placement

Some cells might span more than one column. Cells will wrap to the next row automatically.

<Canvas>
  <Story name="Grid column size and placement">
    <div className="dhig-grid dhig-grid-cols-6 dhig-gap-4">
      <CellContent>Grid cell 1</CellContent>
      <CellContent>Grid cell 2</CellContent>
      <div className="dhig-col-span-2">
        <CellContent className="">Grid cell 3</CellContent>
      </div>
      <CellContent>Grid cell 4</CellContent>
      <CellContent>Grid cell 5</CellContent>
      <CellContent>Grid cell 6</CellContent>
      <div className="dhig-col-start-1 dhig-col-end-7">
        <CellContent>Grid cell 7</CellContent>
      </div>
    </div>
  </Story>
</Canvas>
