# Card

HTML snippets to generate Digital HIG card.

The examples below are intended to illustrate selected variations of card. The interior for content and actions illustrate examples of how the card's interior can be populated, and are not intended to be inclusive of all card interiors.

For more card variations, see [Card component page](/docs/components-card--cover-and-body).
