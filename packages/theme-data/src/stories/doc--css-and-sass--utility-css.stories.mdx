# Utility CSS overview

Digital HIG utility CSS provides a collection of utility classes which extend [Tailwind CSS](https://tailwindcss.com/). All resulting classes are prefixed with the `dhig-`, so `.cursor-pointer` for example becomes `.dhig-cursor-pointer`.

The following Tailwind utility classes have been exposed:

- [alignItems](https://tailwindcss.com/docs/align-items)
