# SASS overview

Here are some of the SASS mixins and their usage.

## Theme mixins

### @dhig-theme

```scss
.dhig-theme--light {
  @include dhig-theme($theme: "light");
}

.dhig-theme--dark {
  @include dhig-theme($theme: "dark");
}
```

## Component mixins

Component mixins can be nested under the theme mixins, passing down either "light" or "dark" respectively.

### @dhig-button--contained--default

```scss
.dhig-button--contained--default {
  @include dhig-button--contained--default($theme: "light");
}
```
