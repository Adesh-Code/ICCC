# Modal

HTML snippets to generate Digital HIG modal. The utility classes `dhig-fixed dhig-inset-0` should be added to the `MuiDialog-root` element for correct positioning.

The backdrop is excluded from this example, but should be placed within the `MuiDialog-root` element, above the `MuiDialog-container` element. This is the HTML pattern for the default backdrop: `<div className="MuiBackdrop-root DhigModal--backdrop--default" aria-hidden="true" />`.

Note: showing the modal open state, the rest of the implementation leading to this state is left to the consumer.
