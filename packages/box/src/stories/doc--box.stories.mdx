The box component is the out-of-the-box MUI [`<Box>`](https://mui.com/components/box/), with Digital HIG spacings.
This component is used to create space between other components in the layout. It also serves as a wrapper component for CSS utility needs.

For a demo implementation of the box component, see the [spacing examples on the MUI website](https://mui.com/system/spacing/).

For related documentation, visit the [spacing page on the Digital HIG website](https://iccc.autodesk.com/spacing/).

## Digital HIG spacing configuration

For the full range of spacing options, see the [spacing primitives page](/story/iccc-theme-data-primitives-spacing--page).

### Margin

The box component can be given margin spacing by using the `m` prop. This applies spacing values to all four sides.

The box component can be given padding spacing by using the `p` prop. This applies spacing values to all four sides.
