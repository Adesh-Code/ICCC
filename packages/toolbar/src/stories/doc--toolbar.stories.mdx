The toolbar component is for use with the [Panel List component](/docs/components-panel-list--batch-processing). It applies styles to the MUI [`<Toolbar>`](https://mui.com/material-ui/api/toolbar/) element.

For related design documentation, please see the [Digital HIG Figma UI Toolkit](https://www.figma.com/file/NpV47EII6zCxkpPoChejg0/%E2%9D%96-iccc-UI-toolkit?node-id=11682%3A277654).

## API

- For all supported MUI `<Toolbar />` props, see [Toolbar API](https://mui.com/material-ui/api/toolbar/).
- For more information about the Digital HIG `<Button />` component, see [the button component page](/docs/components-button--api).
- For more information about the Digital HIG `<IconButton />` component, see [the icon button component page](/docs/components-icon-button--api).
- For all supported MUI `<SvgIcon />` props, see [SvgIcon API](https://mui.com/material-ui/api/svg-icon/).
- For more information about the Digital HIG `<Typography />` component, see [the typography component page](/docs/components-typography--api).

See the [panel list component page](/docs/components-panel-list--batch-processing) for a live example of the toolbar component.

The content of the toolbar is custom. The theming accounts for text, [Digital HIG icons](/docs/components-icon--display-icons), and the dense sizes of both the [Digital HIG icon button](/docs/components-icon-button--api) and [Digital HIG outlined button](/docs/components-button--api).
