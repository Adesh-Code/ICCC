The Card component is a thin wrapper around MUI [`<Card>`](https://mui.com/material-ui/api/card/), with Digital HIG defaults. Included with Card are the nested wrapped components `<CardActions />`, `<CardContent />`, `<CardDivider />`, and `<CardMedia />`.

For related design documentation, please see the [Digital HIG Figma Toolkit](https://www.figma.com/file/Eks986gkqncHjkLnwQjO7g/iccc-sticker-sheets?node-id=4203%3A250690&t=pTfzpGDpokVB8t0o-1).

## API

- For all supported MUI `<Card />` props, see [Card API](https://mui.com/material-ui/api/card/)
- For all supported MUI `<CardMedia />` props, see [CardMedia API](https://mui.com/material-ui/api/card-media/)
- For all supported MUI `<CardContent />` props, see [CardContent API](https://mui.com/material-ui/api/card-content/)
- For all supported MUI `<CardActions />` props, see [CardActions API](https://mui.com/material-ui/api/card-actions/)

### Additional component APIs

Included with Card is component `<CardDivider/>` API. It is a special variation of the `<Divider />` component which provides vertical spacing to separate content from actions. See [Divider docs](/docs/components-divider--api) for more info, and see all examples on this page for utilization.

```jsx
import { CardDivider } from "@iccc/card";
```
