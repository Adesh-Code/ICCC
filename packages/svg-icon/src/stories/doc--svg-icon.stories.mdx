Icon sizes can be set using the `fontSize` property. Supported options are `small`, `medium`, `large` and `inherit` (default).

## Accessibility

The `aria-hidden=true` attribute is automatically added by MUI so that your icons are properly accessible (invisible). This means there is no additional action needed for decorative icons.

For semantic icons, include the `titleAccess` prop with a meaningful value. This will ensure that the necessary `role="img"` attribute and `<title>` element are added.

<Canvas>
  <Story name="Accessibility">
    <SvgIcon fontSize="large" titleAccess="folder icon">
      <AddFolderIcon />
    </SvgIcon>
  </Story>
</Canvas>

For more details about accessibility for SvgIcon, visit the [MUI SvgIcon Accessibility section](https://mui.com/material-ui/icons/#accessibility).
