The Pagination component is used to navigate through a list of items. It applies Digital HIG styles to MUI [`<Pagination>`](https://mui.com/material-ui/react-pagination/).

For related design documentation, please see the Digital HIG [Figma UI Toolkit](https://www.figma.com/file/Eks986gkqncHjkLnwQjO7g/iccc-sticker-sheets?node-id=4203%3A275392&t=iL4WgXwqDMZvxUAw-1).

For the initial engineering prototype, see the archived [Pagination prototype page](https://storybook.digital-help-dev.autodesk.com/iccc/prerelease/PR-435/index.html?path=/docs/components-pagination--default-story).

## API

- For all supported MUI `<Pagination />` props, see [Pagination API](https://mui.com/material-ui/api/pagination/).
- For all supported MUI `<PaginationItem />` props, see [PaginationItem API](https://mui.com/material-ui/api/pagination-item/).

The text under pagination is provided by consumers, as it would be tied to whatever component the pagination is controlling (e.g. table).

## Pagination variants

<VariantRangeNotice />

### Desktop

<Canvas>
  <Story name="Desktop">
    <>
      <Pagination count={10} />
      <Typography variant="smallprint" align="center">
        151-200 of 500 users
      </Typography>
    </>
  </Story>
</Canvas>

### Mobile

Set the `siblingCount` attribute to 0 to ensure the control fits in small screens.

<Canvas>
  <Story name="Mobile">
    <>
      <Pagination count={10} siblingCount={0} />
      <Typography variant="smallprint" align="center">
        151-200 of 500 users
      </Typography>
    </>
  </Story>
</Canvas>

### Disabled

Add the attribute `disabled` to disable the component.

<Canvas>
  <Story name="Disabled">
    <>
      <Pagination count={10} disabled />
      <Typography variant="smallprint" align="center">
        151-200 of 500 users
      </Typography>
    </>
  </Story>
</Canvas>

### Single page

Add the attributes `single` and `disabled` to apply the correct styles when a single page is displayed.

<Canvas>
  <Story name="Single page">
    <>
      <Pagination count={1} single="true" disabled />
      <Typography variant="smallprint" align="center">
        0-9 of 9 users
      </Typography>
    </>
  </Story>
</Canvas>
