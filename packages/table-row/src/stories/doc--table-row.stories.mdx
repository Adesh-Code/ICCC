The table row component is a wrapper around MUI [`<TableRow>`](https://mui.com/material-ui/api/table-row/), with Digital HIG defaults.

For related design documentation, please see the [Digital HIG Figma UI Toolkit](https://www.figma.com/file/NpV47EII6zCxkpPoChejg0/%E2%9D%96-iccc-UI-toolkit?node-id=9009%3A200194).

For usage of table row in the context of `<Table />`, see [table component page](/docs/components-table--api).

## Variants

<VariantRangeNotice />

### Enable interactions

Set `enableInteractions` to `false` to disable hover/active/focus styles.

<Canvas>
  <Story name="Enable interactions">
    {(args) => (
      <Table>
        <TableBody>
          <TableRow>
            <TableCell>Interactions enabled (default)</TableCell>
          </TableRow>
          <TableRow enableInteractions={false}>
            <TableCell>Interactions disabled</TableCell>
          </TableRow>
        </TableBody>
      </Table>
    )}
  </Story>
</Canvas>

### Activated

Use the Digital HIG prop `activated` to render a activated row. This state is only for use when composing [a panel list](/docs/components-panel-list--api).

Note that activated rows always have interactions disabled.

<Canvas>
  <Story name="Activated">
    <Table>
      <TableBody>
        <TableRow activated enableInteractions={false}>
          <TableCell>Cell content</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  </Story>
</Canvas>

### Selected

Use the MUI prop `selected` to render a selected row.

<Canvas>
  <Story name="Selected">
    <Table>
      <TableBody>
        <TableRow selected enableInteractions={false}>
          <TableCell padding="checkbox">
            <FormControlLabel control={<Checkbox checked />} />
          </TableCell>
          <TableCell>Cell content</TableCell>
        </TableRow>
        <TableRow selected>
          <TableCell padding="checkbox">
            <FormControlLabel control={<Checkbox checked />} />
          </TableCell>
          <TableCell>Interactions enabled</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  </Story>
</Canvas>

### Highlighted

Use the Digital HIG prop `highlighted` to render a highlighted row. This state can be used to indicate [new rows in a panel list](/docs/components-panel-list--new-story).

<Canvas>
  <Story name="Highlighted">
    <Table>
      <TableBody>
        <TableRow highlighted enableInteractions={false}>
          <TableCell>Cell content</TableCell>
        </TableRow>
        <TableRow highlighted>
          <TableCell>Interactions enabled</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  </Story>
</Canvas>

</Canvas>
