The tree item component is a thin wrapper around MUI [`<TreeItem>`](https://mui.com/material-ui/api/tree-item/), with Digital HIG defaults.

The `<TreeItem>` component is used to display the contents of a tree node inside `<TreeView />`. It can be used to display a single node or a group of nodes.

For related design documentation, please see the Digital HIG [Figma UI Toolkit](https://www.figma.com/file/Eks986gkqncHjkLnwQjO7g/iccc-sticker-sheets?node-id=4246%3A314582&t=iL4WgXwqDMZvxUAw-1).

For usage of tree item in the context of `<TreeView />`, see [tree view component page](/docs/components-tree-view--variant-minimal).
